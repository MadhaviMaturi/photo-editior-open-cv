# 📸 AI-Powered Photo Editor

A lightweight and interactive **web-based photo editing application** built with **Python, OpenCV, and Streamlit**. This project enables users to enhance images using computer vision techniques through a simple and intuitive interface—without requiring professional image editing software.

> **Built with:** Python • OpenCV • Streamlit • NumPy • Pillow

---

## 🌟 Overview

The AI-Powered Photo Editor allows users to upload an image, apply multiple enhancement filters and creative effects, preview the output in real time, and download the final edited image.

This project demonstrates the integration of **Computer Vision** and **Web Application Development**, making it an excellent portfolio project for showcasing practical AI and image processing skills.

---

## ✨ Features

### Image Enhancement
- Brightness Adjustment
- Contrast Enhancement
- Grayscale Conversion
- Gaussian Blur
- Warm Tone Filter
- Image Sharpening
- Portrait Background Blur

### Creative Effects
- Edge Detection
- Pencil Sketch
- Cartoon Effect

### User Experience
- Upload images (JPG, JPEG, PNG)
- Real-time image preview
- Interactive controls
- Download edited images
- Responsive Streamlit interface

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/photo-editor.git
```

### Move into the Project Folder

```bash
cd photo-editor
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
